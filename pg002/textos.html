Artimanhas da HTML 5

Com meus estudos na área acabei descobrindo muita coisa legal! Uma delas são as tags, que cá entre nós, é um dos charmes da HTML.

  1. Excluindo

    2. Umas dessas funcionalidades é essa. Como você pode observar kkkk a palavra "funcionalidade" foi cortata, ou excluido. Apesar de ainda ser possivel de ler, mas não é mais considerada. Isso foi possivel com a tag <del>.

  2. Destacando

    3. Mais uma kkkk temos o destaque que deixa uma frase destacada em relação ao texto, algo bem util. Usando a tag <ins>

  3. Marcando

    4. Todo mundo ja viu ou usou um marca texto na vida, com certeza você já teve contato, mas você sabia que até na HTML é possivel usar um marca texto? Usando a tag <mark> Um pequeno detalhe nesta parte, podemos mudar a cor do marca texto com CSS. Usando tanto na propria tag (mark) ou na aba "head" do código. Abra o código para ver como se faz!

  4. Sobrescrevendo

    5. Vamos entrar no mundo dos nerds agora kkkkk vou mostrar como organizar numeros de equações. por exemplo: "2x+3=0" isso era pra ser uma equação, mas como vc pode ver não esta correto, matematicamente. Agora veja a diferença: "2x+3=0". Usando a tag <sup> no valor que deseja sovrescrever.

  5. Subscrevendo

    6. Quimica... uma materia que eu amo kkkk e tenho certeza que você ja viu alguma formula, como a da água: H20. Mas espera ai, tem algo de estranho ai. Só um segundo... H20 pronto! Agora ta bonito. Usando a tag <sub> no valor que deseja subscrever.

  6. Texto Grande

    7. Letras grandes ajudam na compreenção do contexto chave do texto, ou apenas deixa ele mais bonito kkkkk. Usando a tag <big> para almentar a fonte do texto.

  7. Texto Pequeno

    8. Minimalismo é um tendencia do mundo moderno. São tantas informações que precisamos diminuir a intencidade, até nos textos. Usando a tag <small> para diminuir a fonte do texto.
