Formatações em HTML

Colocando códigos na pagina

Como a nova verssão da HTML5 se baseia na semântica e não mais no formato, muitas tecnologias foram atualizadas para essa nova verssão, assim aumentando a compatibilidade com tacnicas de S.E.O. Assim muitos se beneficiam com tais ferramentas.

Vamos citar algumas tags de formatação:

  • <h1> até <h6>

    • Essa tag é a famosa "Hierarquia de Titulo." Ela define a ordem semantica que os navegadores e motores de busca rião utilizar para tecnicas de SEO, o que pode fazer diferença na hora dos motores de busca, como o Google, compartilharem os sites. Pois eles usam SEO para exibri os sites que mais são compativeis com esta tecnica

  • <p>

    • Esta tag é a famosa tag paragrafo. Ela define em qual parte sera inserido um paragravo no seu código HTML. Esse texto que vc esta lendo esta entra uma tag <p>.
